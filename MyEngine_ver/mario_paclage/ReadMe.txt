●開発環境
・プログラミングツール：VisualStudio2019
・エンジン：DirectX9
・GitHub：https://github.com/Kasugai0083/MarioMaker
　
※DirectX9を導入していない場合はインストーラーを実行して、
　導入してください。

●概要
　個人制作のアクションゲームです。現在、制作途中。
　残りの実装項目として

　・敵の実装
　・ツール作成
　・各種アニメーション

　などが残っています。

●ファイル内訳
　Csv => マップのデータ
　Res => ゲーム実行に必要なリソースファイル
　Src => ソースコード
　MarioMaker_ME.exe => 実行ファイル

●操作方法
　A/D 	=> プレイヤーの移動
　SPACE => ジャンプ(ショートジャンプ可能)　
　ESC	=> ゲーム終了

●終了条件
　フラッグに触れた時点で終了します。
